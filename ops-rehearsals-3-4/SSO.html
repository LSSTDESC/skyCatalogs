Solar System Object (SSO) quantities for ops-rehearsals 3 and 4

Data are partitioned by (nside=32, ring ordering) healpixel. For each pixel there is a so-called “main file” and a flux file. The latter contains for each object only fluxes for lsst bands, the object id and the time (mjd) of the observation. The latter two fields may be used to join with the main file.
